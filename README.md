# Front End Repo

## Technologies Used
[Vue](https://vuejs.org/)
[Vuetify](https://vuetifyjs.com/en/)
[Tacklebox](https://docs.narrative.dev/tacklebox)
## Project setup

To Install all dependencies
```
yarn
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
